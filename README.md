## 字符串工具箱 - 纯前端版
这是一个完全在浏览器端运行的字符串处理工具箱，无需联网即可使用大部分功能。

![](https://img.runnable.run/blog/7a358a9bd0813ebf2fffa564eeb2e4ce.png)

## 功能特性
- 基础文本处理
- 转义序列转换：\n 等转义字符 ↔ 真实换行符
- 换行符处理：去除换行、规范化换行符（\r\n → \n）
- 字符串转义与反转义（JS/JSON 格式）
- HTML/XML 转义与反转义
- 换行符与 <br> 标签互转 
## 数据格式化
- JSON 格式化（美化/紧凑模式）
- YAML 格式化（美化/紧凑模式）
- XML 格式化（美化/紧凑模式）
- SQL 简单美化（关键字缩进）
## 编码解码
- Base64 编码与解码
- URL 编码与解码
## 文本变换
- 大小写转换
- 首尾空白字符去除（Trim）
- 按行排序（支持去重选项）
- 文本统计（字符数、行数、唯一行数）
- 关键字行提取
